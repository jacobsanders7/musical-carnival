Below is my idea for a new interface on medium.com. Of course with it being just an idea for the website, there are some aspects missing. Instead of hidden navigation where the user has to click on the profile tab to access the different tabs, the tabs could be placed at the top of the website in horizontal form so the user sees them, and has easy access to them. In the actual format of the website, the tabs could be placed above the pre-existing ones, to add extra and easier navigation for the user. Instead of just text fields, the options that appear when the tabs are clicked would be links in the finished product.
